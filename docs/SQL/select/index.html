SELECT

Format:

SELECT [WHAT] FROM [TABLE] (WHERE [CONDITIONS]) (LIMIT n);

In the statement above anything within parentheses is optional.

LIMIT 10;

Order by last_name.

SELECT *

Using ORDER can take one of two keywords for sorting: ASC (default) and DESC.

SELECT * 
FROM `actor`
WHERE `last_name` = "WAHLBERG"
ORDER BY `first_name` DESC;

Count records in the table.

SELECT COUNT(*) 
FROM `actor`;

Count records grouped by last_name.

SELECT `last_name`,

FROM `actor` 
GROUP BY `last_name`

COUNT(*) is the variable name which may not be friendly to some programming languages or additional SQL statements. You can use an ALIAS to change the name of that variable.

SELECT `last_name`, 
  COUNT(*) AS `n`
FROM `actor`
GROUP BY `last_name` 
LIMIT 10;

The HAVING statement can help us filter additionally within the groups.

Find all last_name that has at least three n:

SELECT `last_name`, 
  COUNT(*) AS `n`
FROM `actor`
GROUP BY `last_name`
HAVING `n` >= 3;

When using GROUP you must be careful when adding other variables to your SELECT clause. For example, using the same query above to also include first_name will return bad data.

# Limit 1 record
SELECT `first_name`, 
  `last_name`, 
  COUNT(*) AS `n`
FROM `actor`
GROUP BY `last_name`
HAVING `n` >= 3
LIMIT 1;
Table 8: 1 records
first_name last_name n
CHRISTIAN AKROYD 3

The results above can indicate that we have three ‘CHRISTIAN AKROYD’ values in our table. This isn’t true.

SELECT *
FROM `actor`
WHERE `first_name` = "CHRISTIAN" 
  AND `last_name` = "AKROYD";
