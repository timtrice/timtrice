Combine Two Tables

Oct 10, 2017 3 min read SQL, LeetCode

Problem

Table Person:

+-------------+---------+
| Column Name | Type    |
+-------------+---------+
| PersonId    | int     |
| FirstName   | varchar |
| LastName    | varchar |
+-------------+---------+

PersonId is the primary key column for this table.

Table Address:

+-------------+---------+
| Column Name | Type    |
+-------------+---------+
| AddressId   | int     |
| PersonId    | int     |
| City        | varchar |
| State       | varchar |
+-------------+---------+

AddressId is the primary key column for this table.

Write a SQL query for a report that provides the following information for each person in the Person table, regardless if there is an address for each of those people:

FirstName, LastName, City, State

Table Setup

CREATE DATABASE IF NOT EXISTS `LeetCode`;
USE `LeetCode`;
CREATE TABLE IF NOT EXISTS `Person` (
  `PersonId` INT,
  `FirstName` VARCHAR(255), 
  `LastName` VARCHAR(255)
) ENGINE = InnoDB DEFAULT CHARSET = latin1;
TRUNCATE `Person`;
INSERT INTO `Person`
  (`PersonId`, `LastName`, `FirstName`)
VALUES
  (1, "Wang", "Allen");
CREATE TABLE IF NOT EXISTS `Address` (
  `AddressId` INT, 
  `PersonId` INT, 
  `City` VARCHAR(255), 
  `State` VARCHAR(255)
) ENGINE = InnoDB DEFAULT CHARSET = latin1;
TRUNCATE `Address`;
INSERT INTO `Address`
  (`AddressId`, `PersonId`, `City`, `State`)
VALUES
  (1, 2, "New York City", "New York");

Solution

SELECT `Person`.`FirstName`, 
  `Person`.`LastName`, 
  `Address`.`City`, 
  `Address`.`State`
FROM `Person`
LEFT JOIN `Address`
ON `Person`.`PersonId` = `Address`.`PersonId`;
Table 5: 1 records
FirstName LastName City State
Allen Wang NA NA

Explanation

It’s easy to quickly assume this is a simple LEFT JOIN statement; and that will work. But, we have to notice the key phrase,

regardless if there is an address for each of these people.

Why does that matter? Because, if we do not order the tables properly then we will not quantify this statement. We have to make Person our primary table and LEFT JOIN Addressto ensure we get every record ofPerson, regardless if they have a correspondingAddress` or not.

Cleanup

DROP TABLE `Person`;
DROP TABLE `Address`;
