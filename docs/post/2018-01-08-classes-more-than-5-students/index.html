Classes More Than 5 Students

Jan 8, 2018 2 min read SQL, LeetCode

There is a table courses with columns: student and class.

Please list out all classes which have more than or equal to 5 students.

For example, the table:

+---------+------------+
| student | class      |
+---------+------------+
| A       | Math       |
| B       | English    |
| C       | Math       |
| D       | Biology    |
| E       | Math       |
| F       | Computer   |
| G       | Math       |
| H       | Math       |
| I       | Math       |
+---------+------------+

Should output:

+---------+
| class   |
+---------+
| Math    |
+---------+

Note: The students should not be counted duplicate in each course.

Table Setup

CREATE DATABASE IF NOT EXISTS `LeetCode`;
USE `LeetCode`;
CREATE TABLE IF NOT EXISTS `courses` (
  `student` VARCHAR(255),
  `class` VARCHAR(255)
);
TRUNCATE TABLE `courses`;
INSERT INTO `courses` 
  (`student`, `class`) 
VALUES
  ('A', 'Math'), 
  ('B', 'English'), 
  ('C', 'Math'), 
  ('D', 'Biology'), 
  ('E', 'Math'), 
  ('F', 'Computer'), 
  ('G', 'Math'), 
  ('H', 'Math'), 
  ('I', 'Math');

Solution

SELECT `class` 
FROM `courses`
GROUP BY `class` 
HAVING COUNT(DISTINCT `student`) >= 5;
Table 3: 1 records
class
Math

Explanation

This is a semi-simple query where we just GROUP class then COUNT DISTINCT student within each group.

HAVING must be used here rather than WHERE as we are performing a calculation within the condition. Another alternative would be to use a subquery but this would be slightly slower.

Cleanup

DROP TABLE `courses`;
